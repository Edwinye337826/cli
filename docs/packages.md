# 常用模块

## @w6s

`@w6s` 为团队所使用的默认 [npm-scope](https://docs.npmjs.com/misc/scope), 所有以团队名义发布到 npm 的包，名字都应该以 @w6s/[package-name] 的方式命名！如：`@w6s/fedd`。
